# atproto-heatmap

Visualize your AT Protocol activity across different platforms.

## Supported Collections

Currently supports the following AT Protocol collections (see [lib/collections.ts](lib/collections.ts) for details):

- Bluesky posts
- Leaflet documents
- WhiteWind blog entries
- Smoke Signal events
- Frontpage posts

If a new AT Protocol collection is added and you'd like to see it supported, please raise an issue.
